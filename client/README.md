# \<slr\>

Service Level Report User Interface

## Start API Mock nodejs server

```
$ cd ../mock
$ npm i
$ npm start
```

## Start Polymer dev server

```
$ npm run-script mock
```

## Building the App

```
$ npm build
```

This will create builds of your application in the `build/` directory, optimized to be served in production.

## Running Tests

```
$ npm test
```
